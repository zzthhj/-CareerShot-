## 🇺🇸 English

### 1. Purpose
CareerShot is designed to create professional headshots and profile photos from photos that users capture or select.

### 2. Data Processing Overview
CareerShot primarily processes images on-device. We do not operate a cloud pipeline for photo or face-analysis uploads.

### 3. Face Data
To provide portrait editing features, CareerShot may process face-related data from user-selected photos on-device, including:
- face bounding box
- facial landmarks (for example, eye/lip/chin contour points)
- face angle/pose information (for quality checks)

CareerShot does **not** use face data for identity verification, user profiling, or advertising.

### 4. How Face Data Is Used
Face-related processing is used only to:
- check whether a valid single frontal face is present
- improve portrait crop/composition
- improve portrait rendering/alignment quality (such as portrait and outfit fitting)

### 5. Sharing and Disclosure
We do not sell, share, or disclose face data to third parties.  
Face-analysis metadata is not transmitted to our servers.

### 6. Storage and Retention
- Face-analysis metadata is temporary processing data and is not retained as a separate biometric profile.
- Temporary face-analysis metadata is discarded after processing.
- Final edited photos are saved only when the user explicitly chooses to save/export (for example, to Photos).

### 7. Camera and Photo Library Permissions
- **Camera access** is used to capture photos for editing (for example, background replacement and portrait enhancement).
- **Photo Library access** is used to select photos and save exported images.
- Processing is performed on-device.
